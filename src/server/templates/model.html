How the model works
A simple POC for using bokeh to visualize modelled covid-19 data. Based on bokeh weather example.
There two main components: charts — using bokeh serve to serve bokeh charts in src/charts directory. webserver — a webserver using fastapi to render templates and embeds bokeh charts via the running charts bokeh server using JS scripts. That means that when you access the index site of the fastapi server, it generates the tag for the relevant chart.
